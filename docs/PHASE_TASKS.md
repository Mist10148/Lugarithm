## Status markers

`[x]` implemented & verified (EditMode + in-editor) · `[-]` implemented, needs verification/balancing
· `[ ]` not started · `[!]` blocked by a decision/dependency.

Do not mark runtime work `[x]` from screenshots alone — record the test/observation in the task.

## Definition of Done

Every completed task: satisfies its mapped PRD requirement + README promise; keeps the **two modes
1:1**; uses typed C# with the shared sim/fare/generation as source of truth; keeps authored/heritage
content in data and sourced (folklore labelled); has focused **EditMode** tests where logic exists;
**regenerates affected scenes** via the editor builders; includes one in-editor manual check; updates
[`AI_DISCLOSURE.md`](AI_DISCLOSURE.md) for new AI tools/assets/runtime work; commits no secrets.

## Implemented to date (2026-06-27)

Built and largely verified (EditMode suite green):

- **Foundation:** Bootstrap + persistent managers (GameManager, SettingsManager, LocalizationManager,
  SceneTransitionManager, BadgeUnlockManager, Almanac), JSON save/load, scene transitions. Scenes are
  generated by editor scripts (`Lugarithm → Build All Scenes`).
- **Town hubs:** top-down walk-around per level with ambient NPC dialogue (E), board to drive.
- **Manual Mode:** real-time driving, lane changes, Hold/Toggle brake, Coin Drawer fares, breakdown
  minigames, procedural roadside dressing, dulog markers, dialogue, completion → reveal → results.
- **Automation Mode:** block + code editors, run/step/pause/reset/speed, grid sim + pathfinding,
  one-click Autopilot, mid-drive progression gate, completion flow.
- **Para language:** sequencing, conditionals, loops, `repeat`, `break`/`continue`, **user-defined
  functions**, variables/data structures, built-ins; plain-language error coaching.
- **AI:** Living Story dialogue, Heritage Oracle (Almanac/RAG), Coding Mentor, Co-Pilot/Vibe-Coding
  (Auto/Ask/Plan/Agent/Refactor), context-aware placement — all via Gemini with authored fallbacks.
- **Progression:** per-town badge reveal (placeholder art), currency, gacha.
- **Settings/localization:** sectioned settings with pill selectors; English/Filipino UI live switch.

## Verification commands

- **EditMode tests:** Unity Test Runner (EditMode), or batchmode
  `Unity -runTests -projectPath . -testPlatform EditMode -testResults results.xml`.
- **Scenes:** `Lugarithm → Build All Scenes` (editor menu) after any serialized-wiring change.
- **Secrets:** confirm the root `.env` is git-ignored and untracked (`git status`, `git ls-files`).
- **Manual gate:** in-editor play of the relevant leg(s) in both modes.
